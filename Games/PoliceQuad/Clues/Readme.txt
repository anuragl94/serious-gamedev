Clues are defined with certain properties and values of that property.
Clues contain language based text.
Combinations of logically acceptable clues can be used to create questions.
Example Format : 
	"clue": [
    { "property": "straightSides = 1" },
    { "English": "exactly one straight side" },
    { "Hindi": "केवल एक भुज" },
    { "Kannada": "ನಿಖರವಾಗಿ ಒಂದು ನೇರ ಅಡ್ಡ" },
    { "Telugu": "సరిగ్గా ఒక నేరుగా వైపు" }
  ]
